# MiniCAD

一个基于 C++ 实现的简化三维 CAD 系统，用于探索几何建模原理与 CAD 内核的核心机制。

---

## 🚀 项目简介

MiniCAD 是一个轻量级工程项目，目标是从底层理解 CAD 系统的核心组成，包括：

* 三维几何建模流程
* 拓扑数据结构（B-Rep）
* 基本实体建模与布尔运算
* 渲染与数据结构的结合

本项目更关注 **建模核心与系统设计**，而非 UI 界面复杂度。

---

## 🛠 技术栈

* **C++17**
* **Qt（Widgets）**
* **OpenGL（渲染）**
* **CMake（构建系统）**

---

## 📦 项目结构

```bash
MiniCAD/
 ├── src/
 │    ├── main.cpp
 │    ├── render/      # 渲染模块（OpenGL / 相机）
 │    ├── core/        # 几何核心（B-Rep / 拓扑结构）
 │    ├── modeling/    # 建模功能（基本体 / 布尔运算）
 │    └── ui/          # 界面层（Qt）
 ├── CMakeLists.txt
 └── README.md
```

---

## 🎯 功能规划

* [ ] 基础几何体（立方体、圆柱体）
* [ ] 基于 B-Rep 的拓扑结构实现
* [ ] 简化布尔运算（并 / 差）
* [ ] 三维视图交互（旋转 / 缩放）
* [ ] 基础性能优化（AABB / 空间索引）

---

## 🧠 学习目标

* 理解 CAD 内核如何组织和表示三维模型
* 从零实现基础建模逻辑
* 探索几何计算中的性能优化方法

---

## ⚙️ 编译与运行

```bash
mkdir build
cd build
cmake ..
cmake --build .
./MiniCAD
```
